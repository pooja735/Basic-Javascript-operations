JavaScript Basics Examples

1. Type Conversion

let num = "123";  
let converted = Number(num);

2. Operators (Logical)

let a = true, b = false;  
let result = a && b;

3. if Statement

let age = 20;  
if (age >= 18) { "Adult"; }

4. while Loop

let i = 0;  
while (i < 3) { i++; }

5. for Loop

for (let i=0; i<3; i++) { ... }

6. for...in Loop

let obj = {a:1, b:2};  
for (let key in obj) { ... }

7. for...of Loop

let arr = [10,20,30];  
for (let val of arr) { ... }

8. switch case

let day = 2;  
switch(day) { case 1: ... }

9. Functions

function greet(name) { return "Hello " + name; }
